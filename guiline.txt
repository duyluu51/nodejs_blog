I. Auth router (/auth/...)
    1. POST: /auth/register:
        Body {
            username:string,
            password:string
        }
    2. POST: /auth/login:
        Body {
            username:string,
            password:string
        }
    3. POST: /auth/refresh:
            headers {
                x_authorization:accessToken(string),
            }

            Body {
            refreshToken:string,
            }

II. Student (/student/...)
    1. Get: /student/getStudent:
        headers {
                    x_authorization:accessToken(string),
                }
        Body {
            name
            class
            school
            phoneNumber
            shift
            year
        }
    2. Post: /student/create:
        Body {
            name
            class
            school
            phoneNumber
            shift
            year
        }
    3. Put: /student/update/:id:
        headers {
                    x_authorization:accessToken(string),
                }
        Body {
            name
            class
            school
            phoneNumber
        }
